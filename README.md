# a4b.ai Documentation

Developer documentation for [a4b.ai](https://a4b.ai) — hosted at [docs.a4b.ai](https://docs.a4b.ai).

Built with [Jekyll](https://jekyllrb.com/) and [Just the Docs](https://just-the-docs.com/), deployed via GitHub Pages.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open [http://localhost:4000](http://localhost:4000).
